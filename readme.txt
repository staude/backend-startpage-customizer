Redirect the user after login to a predetermined site in the backend.

== Description ==

English:

This plugin redirects the user after the login to a predetermined site. The user may specify in his profiles in a selection as to which side he wants to start after registration (instead of the dashboard)

If they have more translations for this plugin, please send email to frank@staude.net, I will then install.

Entrys in the changelog with numbers means the ticket id on http://bugs.staude.net


Deutsch: 

Dieses Plugin leitet den Benutzer nach der Anmeldung auf eine zuvor festgelegte Seite. Dazu kann der Benutzer in seinen Profile in einer Auswahlbox festlegen, auf welcher Seite er nach der Anmeldung starten möchte (statt dem Dashboard)

Wenn sie weitere Übersetzungen für dieses Plugin haben, bitte per E-Mail an frank@staude.net schicken, ich werde sie dann einbauen.

Einträge im changelog mit Zahlen verweisen auf die Ticket ID auf http://bugs.staude.net

== Installation ==

1. Install the plugin from within the Dashboard or upload the directory `backend-startpage-customizer` and all its contents to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Use the randomizer.
